# GET /v1/models

List all available models across all registered providers.

## TL;DR

```bash
curl https://api.igrisinertial.com/v1/models \
  -H "Authorization: Bearer YOUR_API_KEY"
```

## Request

### Headers

```
Authorization: Bearer YOUR_API_KEY
```

### Query Parameters

All parameters are optional.

| Parameter | Type | Default | Description |
|-----------|------|---------|-------------|
| `provider` | string | null | Filter by provider: `"openai"`, `"anthropic"`, etc. |
| `capability` | string | null | Filter by capability: `"vision"`, `"tools"`, `"streaming"` |

## Examples

### List All Models

```bash
curl https://api.igrisinertial.com/v1/models \
  -H "Authorization: Bearer YOUR_API_KEY"
```

### Filter by Provider

```bash
curl "https://api.igrisinertial.com/v1/models?provider=anthropic" \
  -H "Authorization: Bearer YOUR_API_KEY"
```

**Response:**


### Filter by Capability

```bash
curl "https://api.igrisinertial.com/v1/models?capability=vision" \
  -H "Authorization: Bearer YOUR_API_KEY"
```

**Response:** Only models with vision capability (GPT-4 Turbo, Claude 3.5 Sonnet, etc.)

## Model Object

| Field | Type | Description |
|-------|------|-------------|
| `id` | string | Model identifier (used in chat completions) |
| `object` | string | Always `"model"` |
| `created` | integer | Unix timestamp of model creation |
| `owned_by` | string | Organization that owns the model |
| `provider` | string | Igris Overture provider name |
| `capabilities` | object | Model capabilities and limits |
| `pricing` | object | Cost per 1K tokens |

### Capabilities Object

| Field | Type | Description |
|-------|------|-------------|
| `vision` | boolean | Supports image input |
| `tools` | boolean | Supports function calling |
| `streaming` | boolean | Supports streaming responses |
| `max_tokens` | integer | Maximum output tokens |
| `max_context_window` | integer | Maximum total tokens (input + output) |

### Pricing Object

| Field | Type | Description |
|-------|------|-------------|
| `prompt_tokens_per_1k` | float | USD per 1000 prompt tokens |
| `completion_tokens_per_1k` | float | USD per 1000 completion tokens |
