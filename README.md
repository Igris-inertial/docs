# Igris Documentation

Official MDX documentation for the Igris AI routing platform.

## Structure

- `docs/overture/` - Overture control plane documentation
  - Architecture and design
  - API reference
  - Core features
  - Configuration guides

- `docs/runtime/` - Runtime execution engine documentation
  - Deployment guides
  - Fleet management
  - Performance optimization
  - API reference

## About Igris

Igris is an intelligent AI routing platform that optimizes LLM inference through:
- Multi-provider routing
- Cost optimization
- Hybrid execution (cloud + edge)
- Real-time observability
